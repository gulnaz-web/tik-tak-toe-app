# Крестики-нолики

Веб-приложение для игры в крестики-нолики с современным интерфейсом.

## Технологии

- **Next.js 15** — React-фреймворк
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **React** — UI-библиотека

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)
